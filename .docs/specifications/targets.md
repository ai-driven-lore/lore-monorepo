# Runtime Targets

Lore Packages are runtime agnostic.

Packages MUST NOT contain platform-specific structures.

Examples of platform-specific files:

* SKILL.md
* AGENTS.md
* Claude Skills
* Cursor Agents

These artifacts are generated by the Lore CLI during installation.

Example:

```bash
lore pull m1cloud/azure-architect
```

The CLI detects the current AI runtime and renders the package accordingly.

Supported runtimes:

* Claude
* Cursor
* GitHub Copilot
* Codex
* Gemini

Future runtimes may be added without changing package contents.
