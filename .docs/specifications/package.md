# Package Specification

A Lore Package is the distribution unit used by the platform.

Package identifier:

```text
<namespace>/<name>:<version>
```

Examples:

```text
m1cloud/azure-architect:1.0.0
m1cloud/terraform-reviewer:1.1.0
community/kubernetes-sre:2.0.0
```

Packages are distributed as:

```text
*.lore
```

Example:

```text
azure-architect-1.0.0.lore
```
