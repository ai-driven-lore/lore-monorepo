# ADR-0004 - Runtime Agnostic Packages

Status: Accepted

## Context

Different AI runtimes use different file structures.

Examples:

* Claude uses SKILL.md
* Cursor uses Agent definitions
* Future runtimes may introduce new formats

## Decision

Lore Packages SHALL remain runtime agnostic.

Platform-specific artifacts SHALL be generated by the Lore CLI during installation.

## Consequences

Benefits:

* Future-proof package format
* Vendor independence
* Single package definition

Trade-offs:

* Additional renderer layer inside the CLI
* Runtime detection logic
